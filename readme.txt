# 🌵 Suculentas | Otro Mundo

Proyecto web interactivo para elegir suculentas de forma lúdica y elegante.  
Pensado para ser escalable, rápido y ofrecer una experiencia memorable.

---

## 🚀 Tech stack

- **HTML5** semántico y modular
- **CSS3** con diseño responsive, variables y animaciones
- **JavaScript** vanilla modular (sin frameworks), listo para escalar
- **Core Web Vitals friendly** (performance y UX priorizados)
