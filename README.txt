Instrucciones para usar la app web local de marca de agua IA avanzada

1. Abrir el archivo index.html en un navegador moderno (Chrome recomendado).
2. Clic en "Cargar Logo" para subir tu logo (puede tener fondo, la IA lo elimina).
3. Clic en "Cargar Imagen" para subir la foto donde quieres poner la marca.
4. Clic en "Hacer la Magia" para que el logo se integre automáticamente.
5. Finalmente, clic en "Descargar Imagen" para guardar el resultado.
6. Puedes subir toda esta carpeta a GitHub Pages para uso online.

¡Listo para usar!
